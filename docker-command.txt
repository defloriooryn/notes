=====perintah docker=====

1. docker images / docker image ls
-> untuk menampilkan list images yang sudah di pull

2. docker container ls / docker ps
-> untuk melihat container yang sedang berjalan di docker daemon

3. docker container ls -a / docker ps -a
-> untuk melihat semua container yang ada di docker daemon

4. docker pull <nama_image>:<tag> (tag disini maksudnya versi dari images)
-> untuk mendownload images kedalam docker

5. docker image rm <nama_image:tag>
-> untuk menghapus images dari docker daemon

6. -docker container --create <nama_container> <nama/id_images> 
   -docker container create --name <nama_container> --publish <port_host:port_container> <nama/id_images>
-> untuk membuat container 

7. docker container rm <nama/id_container>
->untuk menhapus container

8. docker container start <nama/id_container>
-> untuk menjalankan container

9. docker container stop <nama/id_container>
-> untuk menghentikan container

10. docker version 
-> untuk mengecek versi dan mengecek apakah sudah running

11. docker container logs <nama/id_container>
-> untuk melihat log aplikasi di container

12. docker container logs -f <nama/id_container> 
-> untuk melihat log aplikasi di container secara real time 

13. docker container exec -i -t <nama/id_container> /bin/bash
-> untuk masuk kedalam container dan mengeksekusi program yang terdapat di dalam container

14. docker container create --name <nama_container> --env KEY="value" --env KEY2="value" <nama/id_images>
-> untuk menambah environment variabel ke dalam container
