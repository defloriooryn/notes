=====perintah docker=====

1. docker images / docker image ls
-> untuk menampilkan list images yang sudah di pull

2. docker container ls / docker ps
-> untuk melihat container yang sedang berjalan di docker daemon

3. docker container ls -a / docker ps -a
-> untuk melihat semua container yang ada di docker daemon

4. docker pull <nama_image>:<tag> (tag disini maksudnya versi dari images)
-> untuk mendownload images kedalam docker

5. docker image rm <nama_image:tag>
-> untuk menghapus images dari docker daemon

6. -docker container --create <nama_container> <nama/id_images> 
   -docker container create --name <nama_container> --publish <port_host:port_container> <nama/id_images>
-> untuk membuat container 

7. docker container rm <nama/id_container>
->untuk menhapus container

8. docker container start <nama/id_container>
-> untuk menjalankan container

9. docker container stop <nama/id_container>
-> untuk menghentikan container

10. docker version 
-> untuk mengecek versi dan mengecek apakah sudah running

11. docker container logs <nama/id_container>
-> untuk melihat log aplikasi di container

12. docker container logs -f <nama/id_container> 
-> untuk melihat log aplikasi di container secara real time 

13. docker container exec -i -t <nama/id_container> /bin/bash
-> untuk masuk kedalam container dan mengeksekusi program yang terdapat di dalam container

14. docker container --env KEY="value" --env KEY2="value" <nama/id_images>
-> untuk menambah environment variabel ke dalam container

15. dockeer container stats
-> untuk menampilkan penggunaan resource seperti cpu/memori di container

16. docker container create --memory <total_memori> --cpu <total_cpu> (contoh --memory 200m / 1g, --cpus 1) <nama/id_images>
-> untuk mengatur penggunaan resource container

17. docker container --mount "type=bind,source=folder,destination=folder,readonly" <nama/id_image>
-> untuk melakukan mounting / sharing file antara host dan container ===> 
      contoh : docker container create --name exmongo --mount "type=bind,source=C:\Users\Nizam\Desktop\notes\tesdocker,destination=/data/db" --publish 27018:27017 --env 
               MONGO_INITDB_ROOT_PASSWORD=nizam --env MONGO_INITDB_ROOT_USERNAME=nizam mongo

18. docker volume ls 
-> untuk menampilkan daftar volume

19. docker volume create <nama_volume> & docker volume rm <nama_volume>
- untuk membuat & menghapus volume

20. docker search <nama_app>
-> untuk mencari app yang tersedia untuk dipull di docker hub

21. docker run -i -t <nama_image_os>
-> untuk membuat sekaligus menjalankan container

22. docker network create --driver <nama_driver> <nama_container> ( contoh --driver bridge jaringan1 )
- untuk membuat network di container

23. docker network ls
-> untuk melihat network yang ada di docker

24. docker network rm <nama_network>
-> untuk menghapus network

25. docker network create --driver <nama_network> <nama_container> <image/id_image>
-> untuk membuat container sekaligus attach ke jaringan yang sudah di buat

//untuk mengupload images ke docker hub menggunakan perintah:
   - docker tag redhat <nama/id_images> defloriooryn/redhat-aja <nama image yang akan di upload>
   - docker push defloriooryn/redhat-aja

26. docker <images/container/network/volume> prune
-> untuk menghapus secara otomatis resource yang tidak digunakan
   
27. docker inspect <nama_images/container/volume/network>
-> untuk melihat detail dari resource 
   
28. docker search <nama_images>
-> untuk search images dari docker hub

29. docker image save -o /c/Desktop/image.tar <nama_image>
-> untuk backup image ke host

30. docker
